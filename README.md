# Task Manager API

## Overview
This is an API for the task manager.

## Technologies
* **Bun:** The runtime environment.
* **Hono:** The web framework.
* **Zod:** Used for validation parameters.
* **PostgreSQL:** The database system.
* **Prisma:** The ORM for database management.
